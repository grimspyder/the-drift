### Installation & Downloads

#### Windows
1. Download `The Drift v{version}.exe`
2. Run the installer
3. Launch from Start Menu or desktop shortcut

#### Linux
1. Download `the-drift-{version}.x86_64`
2. Make executable: `chmod +x the-drift-{version}.x86_64`
3. Run: `./the-drift-{version}.x86_64`

#### macOS
1. Download `The Drift v{version}.dmg`
2. Mount the disk image
3. Drag app to Applications folder
4. Run from Launchpad

### System Requirements

**Windows:**
- Windows 10 or later
- DirectX 11 compatible GPU
- 2GB RAM minimum

**Linux:**
- Ubuntu 20.04 or equivalent (glibc 2.31+)
- OpenGL compatible GPU
- 2GB RAM minimum

**macOS:**
- macOS 10.13 or later
- Metal compatible GPU
